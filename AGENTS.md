# AGENTS.md

## 目标

实现班级日记本第一版，优先保证：

1. 接口和页面闭环
2. 权限正确
3. 匿名展示与管理员追溯正确
4. 本地开发可运行

## 固定技术栈

- 小程序：微信原生 + JavaScript + WXML + WXSS
- 后端：Node.js + Express + Prisma + MySQL + JWT + Multer + axios

## 实现原则

- routes 只做路由绑定
- controllers 只做请求解析和响应
- services 负责业务逻辑
- middleware 负责鉴权、错误处理、上传
- utils 放通用工具
- 不实现教师、公告、AI 审核、Web 管理后台等超范围功能

## 关键权限规则

- 未登录不能访问业务接口
- 未审核通过用户不能访问班级数据
- 非班级管理员不能访问管理接口
- 普通接口不能返回匿名真实作者
- 管理接口可以返回匿名真实作者
- 管理员不能禁言自己、移除自己、取消最后一个管理员
