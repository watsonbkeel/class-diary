# PERMISSION_MODEL

## 角色

- 班级创建者：任何登录用户都可以创建班级，并自动成为该班级管理员
- 班级管理员：班级内管理权限
- 普通成员：班级内容使用权限

## 成员权限

- 已通过审核成员可看班级内容
- 未通过审核成员不能访问班级内容
- 被禁言成员不能发日记、评论、回复

## 管理员权限

- 审核入班申请
- 查看成员列表
- 禁言 / 取消禁言
- 设置 / 取消管理员
- 移除成员
- 查看匿名真实作者
- 隐藏日记 / 评论
- 处理举报

## 限制

- 管理员不能移除自己
- 管理员不能禁言自己
- 不能取消最后一个管理员
- 普通接口不得返回匿名真实作者
