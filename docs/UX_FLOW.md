## 登录/注册流程

1. 打开小程序。
2. 检查本地 token。
3. 无 token 进入登录页。
4. 新用户填写账号名、密码和昵称注册。
5. 老用户使用账号名和密码登录。
6. 登录成功保存 token 和用户信息，进入班级选择页。

当前版本不再依赖微信 `wx.login` 获取身份，避免教育体验版 AppSecret 和跨设备 mock 身份问题。

## 首页班级选择流程

首页使用两个 Tab：

### 班级列表

1. 展示新手指引。
2. 展示全部班级列表。
3. 未加入或可申请的班级优先展示，已加入班级靠后。
4. 用户可直接申请加入班级。
5. 已申请的班级显示审核中。
6. 已加入的班级可直接进入日记流。
7. 仍支持通过邀请码查询班级。

### 我的班级

1. 展示当前用户已加入的班级。
2. 每个班级卡片右侧展示当前身份：`管理员` 或 `成员`。
3. 点击班级卡片进入日记列表。
4. 创建班级入口放在我的班级列表底部。
5. 创建班级成功后，创建者自动成为该班级管理员并进入日记列表。

## 日记互动流程

1. 从班级进入日记列表。
2. 可发布文字/图片日记，支持匿名发布。
3. 可进入日记详情查看图片、评论和回复。
4. 图片可点击全屏预览。
5. 可点赞日记或评论。
6. 可举报日记或评论。
7. 作者可删除自己的日记或评论。

## 通知流程

1. 日记被点赞、评论被点赞、日记被评论、评论被回复、入班申请处理和举报处理会产生通知。
2. 用户从日记列表进入通知页。
3. 通知页支持查看未读、标记单条已读和全部已读。
4. 点击带有关联日记的通知，会先标记已读，再跳转到对应日记详情。
5. 如果通知关联评论，详情页会高亮对应评论或回复。
6. 从详情页返回时，仍回到通知列表，方便继续查看其他通知。

## 我的页面

- 可查看当前用户信息。
- 可修改全局昵称。
- 可修改当前班级昵称。
- 可退出登录并清理本地 token。

## 管理流程

只有 `currentClass.role === 'admin'` 时显示管理入口。

管理员可以：

1. 查看管理首页统计。
2. 审核入班申请。
3. 查看和管理成员。
4. 禁言/取消禁言成员。
5. 设置/取消管理员。
6. 移除成员。
7. 查看并处理举报。
8. 隐藏违规日记或评论。
9. 查看匿名内容真实作者。

## 页面要求

- 每页包含加载态、空状态和错误提示。
- 普通成员不显示管理入口。
- 普通成员不能通过页面路径访问管理功能。
- 匿名内容对普通成员只显示匿名身份，对管理员可追溯真实作者。
