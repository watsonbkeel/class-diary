# IMPLEMENTATION_PLAN

## 当前实现阶段

项目第一版已完成微信原生小程序、Node.js Express 后端、Prisma + MySQL 数据库、Docker Compose MySQL、图片上传、JWT 认证和小程序内管理员管理功能。

当前版本重点能力：

1. 账号名 + 密码注册登录。
2. 公开班级列表和我的班级双 Tab 首页。
3. 任何登录用户都可以创建班级，创建者自动成为管理员。
4. 申请加入班级需要管理员审核。
5. 日记、图片、评论、回复、点赞、举报和通知闭环。
6. 管理员可审核入班、管理成员、隐藏内容、处理举报并追溯匿名真实作者。
7. 图片按 ISO 周目录存储。
8. 通知可跳转到关联日记并高亮评论。

## 已完成阶段

4. 实现管理员审核、成员管理、举报处理、隐藏内容。

## 后续维护优先级

- 优先保证接口和页面闭环。
- 优先保证权限正确和数据隔离。
- 优先保证匿名展示与管理员追溯正确。
- 图片、通知、班级入口等高频流程应优先回归测试。
- 不扩展教师、公告、AI 审核、Web 管理后台等第一版以外功能。
